    Configurable Guided Selling Journey

    Note: This feature is introduced with version 1.0 of the TUA Spartacus libraries.

    Configurable Guided Selling (CGS) is the process in which a customer is taken through a step-by-step guided selling journey to purchase a complex product bundle.

    For more information, see Configurable Guided Selling and Configurable Guided Selling for BPOs on the SAP Help Portal.

    Table of Contents

    • Supported Backend Functionality
    • Configuring Configurable Guided Selling Journey
    • Further Reading

    Supported Backend Functionality

    Configurable Guided Selling functionality relies on the structure of a Bundled Product Offering configured in the Telco and Utilities Accelerator.

    • For more information about Product Catalog and about Bundled Product Offering structure, see TUA Product Catalog in the TUA Help Portal.
    • For more information about the structure of Bundled Product Offerings, see TUA Data Model in the TUA Help Portal.

    Configuring Configurable Guided Selling Journey

    No special configuration is required.
