    Overview

    General Release Information

    New Spartacus libraries are usually published every week. Each release may contain bug fixes, improvements, and new features.

    When new features are available for release, normally we will publish a pre-release “next” version of the libraries with the new features, in order to get feedback. When the final, new, minor version is ready, we usually publish a release candidate (RC), with the final new x.y.0 a few days or weeks later.

    Some of the links provided in the following lists point to SAP Commerce Cloud Accelerator documentation, to give an idea of what the feature is about. These links are for context only. While we strive for feature parity, the Spartacus implementation of features may not work exactly as in Accelerator.

    Other Release Documentation

    • For information about features published in pre-release libraries, see Pre-Release Information.

    Release Date Notes

    3.4 released! See Release Information for All Versions of Spartacus Libraries.

    As of June 23, current plans are:

    • 4.0 (new major): Feature-freeze end of June, final release mid-July.
    • 4.x: Every 6 weeks, new releases with new features end of Q2 and into Q3
    • 5.0: Planned for December 2021

    Q3 2021

    4.0:

    • Angular 12
    • Cleanup of deprecated code and updates to schematics
    • Code split of checkout
    • Blue theme

    4.x or 5.0:

    • B2B Inventory Display - For more information, see the SAP Commerce Cloud Documentation
    • B2B Import Products to Saved Cart and Export from Cart (*) (from/to files) - for more information, see the SAP Commerce Cloud Documentation
    • B2B Quick Order - for more information, see the SAP Commerce Cloud Documentation
    • Cart Validation More formal checking of carts before you proceed to checkout, for example, check if a product is still in stock
    • Image Zoom on Product Details page (will be released in an incubator library)
    • Code splitting for cart
    • One app for running B2C and B2B stores

    5.0:

    • Buy online pickup in store - for more information, see the SAP Commerce Cloud Documentation
    • B2B Commerce Quotes (requires API update planned for 2105) - for more information, see the SAP Commerce Cloud Documentation
    • Accessibility Screen Reader support
    • Site map

    Future Outlook - 2022 or later

    The items in this section are planned for 2022 but do not yet have a specific date. APIs already support these features.

    • Product bundles (requires SAP Commerce Cloud 2011) - for more information, see the SAP Commerce Cloud Documentation
    • ASM customer list - for more information, see the SAP Commerce Cloud Documentation
    • ASM Anonymous Cart Binding (requires API introduced in 2005)
    • ASM Search Autocomplete (requires API introduced in 2005)
    • Potential Promotions
    • Self-service customer support through the Customer Service Module (requires API update planned for 2105) - for more information, see the SAP Commerce Cloud Documentation
    • Buy it again (add to cart from existing order)
    • Social sharing of product information
    • B2B Order Form Builder - for more information, see the SAP Commerce Cloud Documentation

    The following features are planned but depend on API updates, scheduled for the release-after-2105:

    • B2B Re-order
    • B2B Multi-dimensional product support and order grid
    • B2B Registration

    The following features are on our radar but depend on API updates not yet scheduled:

    • Punchout - for more information, see the SAP Commerce Cloud Documentation
    • Captcha - for more information, see the SAP Commerce Cloud Documentation
    • ASM Customer 360° (requires API update not yet planned)
    • B2B Future Stock (requires API update not yet planned) - for more information, see the SAP Commerce Cloud Documentation
    • B2B Account Summary (requires API update not yet planned) for Commerce Org
